📝 AI-Powered Sentiment Analysis DashboardThis project is a high-performance, real-time web application designed to demonstrate the integration of Natural Language Processing (NLP) directly within a browser environment. By utilizing a pre-trained machine learning model, the dashboard analyzes the emotional tone of user-provided text, providing instant feedback on whether the sentiment is positive, negative, or neutral.🚀 Key FeaturesReal-time Processing: Leverages asynchronous JavaScript to analyze text as the user types, ensuring a seamless and interactive experience.Visual Mood Meter: A dynamic CSS-driven gauge that changes color and scale ($0.0$ to $1.0$) based on the model’s confidence score.Privacy-Centric: Since the model runs entirely on the client side via the browser, no user data is sent to a backend server, ensuring total data privacy.Advanced UI: A clean, responsive dashboard built with modern CSS Flexbox to provide a professional user interface.🛠️ Technical StackFrontend Architecture: Developed using semantic HTML5 and CSS3 for a structured, accessible, and aesthetically pleasing layout.Machine Learning Engine: Powered by ml5.js, a high-level library built on top of TensorFlow.js, making complex neural networks accessible for web developers.Model Logic: Utilizes a pre-trained Recurrent Neural Network (RNN) specifically optimized for movie reviews and general sentiment patterns.🧠 System Logic & ArchitectureThe application works by converting string inputs into numeric tokens. These tokens are passed through the RNN, which evaluates the sequence of words to determine the overall "weight" of the sentiment.Positive (Score > 0.7): Indicates highly favorable language.Neutral (0.4 - 0.7): Represents objective or balanced statements.Negative (Score < 0.4): Flags critical or unfavorable text.
